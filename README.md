# Student Management System (Java)

A console-based application developed in Java to efficiently manage student records using object-oriented principles and file handling for persistent data storage.

## Key Features
- Add and store new student records
- View all stored student details
- Search students by unique ID
- Persistent storage using file handling

## Technical Highlights
- Implemented using core Java and OOP concepts (Encapsulation, Classes, Objects)
- Used BufferedReader and BufferedWriter for efficient file operations
- Designed a structured and modular codebase for maintainability
- Menu-driven interface for smooth user interaction

## Project Structure
src/main/
- MainApp.java (Application entry point)
- Student.java (Data model)
- StudentService.java (Business logic)
- FileHandler.java (File operations)

## How to Run
- Import project into Eclipse or any Java IDE
- Run MainApp.java
- Interact with the system via console menu
